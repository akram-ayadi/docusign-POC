## Get started

1. Install dependencies

   ```bash
   npm install
   ```

2. Start the app on iOS simulator
   - Install Xcode
   - Install Xcode Command Line Tools
   - Install an iOS Simulator in Xcode
   - Run:

   ```bash
    npm run ios
   ```


**NOTE**: `DOCUSIGN_URL` in `app/(tabs)/sign.tsx` is the Docusign embedded signing URL.
